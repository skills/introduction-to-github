# Deployment Records

This directory contains deployment records for ScrollVerse ecosystem contracts.

## Contracts

### AnchorManifest
Metadata Manifest Anchoring - Merkle Root and IPFS CID storage.

### ProtocolRegistry (Security Layer 2.0)
Governance registry for ScrollVerse ecosystem protocols with lifecycle management and DAO-enforced transitions.

## File Format

Deployment files are named: `{contract}-{network}-{chainId}.json` or `{network}-{chainId}.json`

Examples:
- `protocol-registry-sepolia-11155111.json`
- `sepolia-11155111.json`

## Schema

### AnchorManifest Schema
```json
{
  "network": "string",
  "chainId": "string", 
  "contract": "string",
  "address": "string",
  "owner": "string",
  "deployedAt": "ISO 8601 timestamp",
  "deployer": "string",
  "constructorArgs": ["array of constructor arguments"],
  "verified": "boolean",
  "etherscanUrl": "string"
}
```

### ProtocolRegistry Schema
```json
{
  "network": "string",
  "chainId": "string",
  "contract": "ProtocolRegistry",
  "address": "string",
  "governor": "string",
  "deployedAt": "ISO 8601 timestamp",
  "deployer": "string",
  "constructorArgs": [],
  "securityLayer": "2.0",
  "postDeploymentSteps": [
    "Bootstrap critical native ScrollVerse protocols (e.g., CHXTOKEN)",
    "Transfer governance to ScrollVerseDAO Timelock Controller"
  ]
}
```
