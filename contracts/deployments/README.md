# Deployment Records

This directory contains deployment records for the AnchorManifest contract.

## File Format

Deployment files are named: `{network}-{chainId}.json`

Example: `sepolia-11155111.json`

## Schema

```json
{
  "network": "string",
  "chainId": "string", 
  "contract": "string",
  "address": "string",
  "owner": "string",
  "deployedAt": "ISO 8601 timestamp",
  "deployer": "string",
  "constructorArgs": ["array of constructor arguments"],
  "verified": "boolean",
  "etherscanUrl": "string"
}
```
